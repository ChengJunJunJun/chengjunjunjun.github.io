#### 🧭 Zsh 常用快捷键清单（默认 Emacs 模式）

| 功能                | 快捷键                      | 说明               |
| ----------------- | ------------------------ | ---------------- |
| 删除光标到行首           | `Ctrl + U`               | 删除光标前的所有字符       |
| 删除光标到行尾           | `Ctrl + K`               | 删除光标后的所有字符       |
| 删除光标前一个单词         | `Ctrl + W`               | 删除前一个单词          |
| 清空整行命令（推荐组合）      | `Ctrl + A` 然后 `Ctrl + K` | 移动到行首并删除到行尾      |
| 移动到行首             | `Ctrl + A`               | 类似 Home 键        |
| 移动到行尾             | `Ctrl + E`               | 类似 End 键         |
| 向前移动一个字符          | `Ctrl + B`               | 类似左方向键           |
| 向后移动一个字符          | `Ctrl + F`               | 类似右方向键           |
| 搜索历史命令（增量搜索）      | `Ctrl + R`               | 交互式搜索历史命令        |
| 取消当前命令            | `Ctrl + C`               | 放弃当前命令行          |
| 清屏（类似 `clear` 命令） | `Ctrl + L`               | 清除屏幕但保留命令行       |
| 撤销上一步编辑           | `Ctrl + _`               | 撤销最近的编辑（有些终端不支持） |
